# 標號程式碼
```

1. 數字標號
2. 數字標號
3. 數字標號
5. 數字標號

(要數字開頭但不想變成項目清單可以加反斜線)
2023\. sysh.

- 其他標號(一次出現)

+ 其他標號

* 其他標號(依序出現)

```
---

    - 無序清單子清單
