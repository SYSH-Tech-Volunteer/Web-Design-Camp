# 第一個網站

打開VSCode開啟一個資料夾，建立一個檔案叫做`index.html`
（當然要取別的名字也可以）並在裡面隨便打點字
我們到檔案總管找到這個檔案並打開，你的第一個網頁完成了

> *網站講白就是一個要用瀏覽器打開的Word檔*

但是只有純文字太無聊了，所以我們在文字間做一些標記讓他們有不一樣的意義

# 無序清單

```html
<ul>
  <li>a</li>
  <li>b</li>
  <li>c</li>
</ul>
```

+ a
+ b
+ c
  
---

# 有序清單

```html
<ol>
  <li>a</li>
  <li>b</li>
  <li>c</li>
</ol>
```

1. a
3. b
2. c
  
---

# 清單裡可以有清單

```html
<ul>
  <li>清單裡可以有清單</li>
  <ul>
    <li>沒錯</li>
    <li>就是這樣</li>
  </ul>
</ul>
```

+ 清單裡可以有清單
  + 沒錯
  + 就是這樣

# 超連結
### 格式

```html

```

```html

```
