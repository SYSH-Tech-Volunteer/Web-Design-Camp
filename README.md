# 2023西苑網頁設計營隊

西苑資訊志工主辦的網頁設計營隊，讓你在短短的兩天內，學會網頁設計的基礎知識，並且在營隊結束後，可以自己架設一個屬於自己的網站。

[Github 倉庫](https://github.com/SYSH-Tech-Volunteer/Web-Design-Camp) | [Discord社群](https://discord.gg/Awwasw45Gc)
